# vita-marginal-packages
A set of retarded 'better late than never' packages for VitaSDK.

- sdl2_vitagl - [Northfear's fork using vitaGL](https://github.com/Northfear/SDL/tree/vitagl) with a probably not needed patch.
- libyaml, sdl2_sound and uchardet explain themselves.

# building/installing
Run `vita-makepkg` in respective directory to build. run `vdpm *-arm.tar.xz` to install | [Copy from Isage](https://github.com/isage/vita-packages-extra)
